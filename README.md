## 📖 About

This repository documents my **entire learning journey** through the [100 Days of Code: The Complete Python Pro Bootcamp](https://www.udemy.com/course/100-days-of-code/) by Dr. Angela Yu on Udemy.

Each folder corresponds to one day of the course, containing practice code, exercises, and the day's final project. This repository will **continue to be updated** until Day 100 is complete.

## 🚀 How to Run

1. Clone this repo:
   ```bash
   git clone https://github.com/lmaothisname/-100-Days-of-Code-The-Complete-Python-Pro-Bootcamp.git
   ```
2. Navigate into the day you want to explore:
   ```bash
   cd "Day 1"
   ```
3. Run the Python file:
   ```bash
   python main.py
   ```

> **Note:** Some projects (Day 32 onwards) require personal API keys. Create a `.env` file and add your keys there.
